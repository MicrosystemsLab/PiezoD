piezoD - piezoresistive and piezoelectric cantilever design optimization

piezoD is a tool for optimizing the sensing performance of piezoresistive and piezoelectric microcantilevers. It is currently written in object oriented Matlab. piezoD is designed to make it fast and easy to design cantilevers.

features

  • materials

    • p-type and n-type single crystal silicon piezoresistive cantilevers

    • aluminum nitride (AlN) and lead zirconate titanate (PZT) piezoelectric cantilevers

  • performance model

    • noise: 1/f, johnson, amplifier

    • sensitivity: concentration dependent piezoresistance factor

    • doping methods: epitaxy, diffusion, including solid solubility limitations

  • constraints available

    • simple bounds: all design parameters, e.g. cantilever length, bias voltage

    • general nonlinear constraints: power dissipation, temperature, resonant frequency (vacuum or fluid), resistance, spring constant

  • optimization targets

    • force resolution

    • displacement resolution
