piezoD - piezoresistive cantilever design optimization

piezoD is a tool for optimizing the performance of piezoresistive cantilevers. It is currently written in object oriented Matlab. piezoD is designed to make it fast and easy to design piezoresistive cantilevers.

features

  • materials

    • p-type and n-type single crystal silicon cantilevers

  • performance model

    • noise: 1/f, johnson, amplifier

    • doping methods: epitaxy, diffusion

  • constraints available

    • cantilever design: dimensions, bias voltage, doping timetemperatureconcentration

    • computed: power dissipation, resonant frequency (vacuum or fluid), resistance

  • optimization targets

    • force sensing in a given frequency range (broadband)
